# SEO

- Target specific long-tail queries a real buyer would type ("free invoice
  tool for freelance photographers"), not broad head terms you'll never rank
  for with a new domain.
- One primary keyword per page; the page title and first paragraph should
  both contain it naturally.
- Content that answers the query completely beats content that's merely
  long. Thin AI-generated filler actively hurts rankings — never ship it.
- Technical baseline: fast load, mobile-friendly, one H1, descriptive meta
  description, semantic HTML.
