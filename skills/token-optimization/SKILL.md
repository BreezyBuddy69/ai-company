# Token Optimization

- Never restate context that's already in the "Relevant long-term memory" or
  "Recent activity" blocks — build on it, don't repeat it.
- Prefer one dense sentence over three loose ones.
- When summarizing, keep concrete numbers, names, and URLs; drop hedging and
  filler ("it's worth noting that...", "in conclusion...").
- If you don't have enough information to act, say exactly what's missing in
  one line rather than speculating at length.
